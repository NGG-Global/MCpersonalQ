# מנהל/ת בעידן ה-AI – שאלון רפלקציה עצמית

מיני-אפליקציית Web לשאלון רפלקציה אישי, בעברית מלאה ובכיוון RTL, מותאמת לדסקטופ ולמובייל.
החישוב והצגת התוצאות מתבצעים כולם בצד הלקוח, ללא איסוף פרטים אישיים וללא שמירת נתונים בשרת.

## יכולות עיקריות

- מסך פתיחה, מסך הנחיות ו-10 מסכי שאלות (שאלה אחת בכל פעם).
- סולם דירוג 1–5, ולידציה לפני מעבר, וניווט קדימה/אחורה עם שמירת הבחירות.
- חישוב אוטומטי של שני ממדי העל (כישורים טכנולוגיים ואנושיים) וחמשת ממדי "כוכב הצפון".
- מסך תוצאות הכולל כרטיסי ציון, גרף עמודות אופקי (וגרף רדאר בדסקטופ) ומשוב מילולי מקורי לפי טווח הציון.
- ייצוא התוצאות כ-PDF (A4) וכ-PNG (1080×1350) דרך רכיב ייעודי בגודל קבוע.

## הרצה

```bash
npm install
npm run dev       # סביבת פיתוח
npm run build     # בנייה לפרודקשן
npm run preview   # תצוגה מקדימה של תיקיית dist
```

## מבנה

```
src/
  data/questionnaire.ts   # תוכן השאלון, המשובים והמיפויים – מקור אמת לתוכן (נעול)
  lib/scoring.ts          # לוגיקת החישוב ובחירת המשוב לפי טווח הציון
  lib/exportPng.ts        # ייצוא PNG (html-to-image)
  lib/exportPdf.ts        # ייצוא PDF (html-to-image + jsPDF, A4 RTL)
  components/             # רכיבי הממשק: מסכים, סולם דירוג, כוכב הצפון, משוב וייצוא
  fonts/                  # גופן Heebo מתארח מקומית (ללא תלות חיצונית, הטמעה תקינה בייצוא)
  index.css               # שפת העיצוב (צבעים, טיפוגרפיה, RTL)
```

## הערות תוכן

כל נוסחי השאלון, ההיגדים והמשובים מופיעים בדיוק כפי שנמסרו, ואין לשנותם.
המידע על שיוך השאלה לממד ולסוג הכישור משמש לחישוב פנימי בלבד ואינו מוצג למשתתפ/ת בזמן המענה.
